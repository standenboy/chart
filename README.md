
# Chart

Takes a ugly chart and makes it pretty.

# Usage

Write a file with a few lines of data, each point seperated by a | charater.

Like so:

x|y|z

256|52|61

6|5|125

run chart on it for it to be nicely formated.

`chart file.chart`
